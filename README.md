
# Ultimate Invite Tracker Bot

Features:
- Invite tracking
- Invite leaderboard
- Invite rewards
- Fake account detection
- Invite giveaways
- Simple web dashboard

Setup:

1. Install Node.js
2. Run:
npm install

3. Edit config.json
Add bot token, client ID, and server IDs.

4. Start bot:
node bot.js

Commands:
/invites
/leaderboard
/invite-giveaway prize:<prize> duration:<time>

Example:
/invite-giveaway prize:Nitro duration:7d
